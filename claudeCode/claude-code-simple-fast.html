🎨 設計 Prompts

# 設計風格探索 claude "為 [產品名稱] 設計現代化的視覺風格： - 目標用戶：[用戶群體] - 產品定位：[商業/專業/創意] - 情感訴求：[信任/活力/創新] 請提供： 1. 色彩配色方案 (主色/輔色/強調色) 2. 字體選擇建議 3. 視覺元素風格 (圓角/陰影/漸層) 4. 參考設計案例" # 組件設計系統 claude "建立設計系統的組件規範： - 按鈕 (Primary/Secondary/Danger) - 表單元素 (Input/Select/Checkbox) - 卡片組件 (基本/互動/資訊) - 導航元素 (頂部/側邊/麵包屑) @ design-requirements.md" # 響應式設計規劃 claude "設計響應式布局方案： - 桌面版 (1200px+) - 平板版 (768px-1199px) - 手機版 (320px-767px) 重點考量： 1. 導航適配方式 2. 內容重新排列 3. 互動方式調整"
