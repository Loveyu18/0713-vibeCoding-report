## 第二部分：Claude Code 概述 (8 分鐘)

### 1. 什麼是 Claude Code？
- **定義：** Anthropic 開發的智能編程助手
- **核心理念：** 將想法快速轉化為程式碼
- **獨特優勢：**
  - 直接在終端機中工作
  - 可編輯檔案、執行指令、建立提交
  - 企業級安全與隱私保護
  - 支援多種程式語言與框架

### 2. 與其他 AI 編程工具的差異
- **GitHub Copilot：** 程式碼補全為主
- **Cursor：** IDE 整合編輯器
- **Claude Code：** 終端機原生，全方位開發助手
- **優勢比較表格**

### 3. 適用場景
- 快速原型開發
- 程式碼重構與優化
- 除錯與問題解決
- 技術文檔撰寫
- 學習新技術與框架

## 第三部分：安裝與設定 (7 分鐘)

### 1. 系統需求
- **前置條件：** Node.js 18 或更高版本
- 支援平台：macOS、Linux、Windows (WSL)
- 終端機或命令提示字元

### 2. 安裝方式 (三選一)
```bash
# 方式一：NPM 安裝
npm install -g @anthropic-ai/claude-code

# 方式二：macOS/Linux 原生安裝
curl -fsSL claude.ai/install.sh | bash

# 方式三：Windows PowerShell
irm https://claude.ai/install.ps1 | iex
```

### 3. 初次設定
- API 金鑰設定
- 專案目錄導航
- 基本組態檔案
- 權限與安全設定

### 4. 驗證安裝
```bash
claude --version
claude --help
```

## 第四部分：核心功能實戰演示 (10 分鐘)

### 1. 基本指令操作
```bash
# 啟動互動模式
claude

# 單次任務執行
claude "建立一個 React 元件"

# 查詢模式
claude -p "這個專案使用什麼技術？"

# Git 提交
claude commit
```

### 2. 核心功能展示 (Live Demo)

#### A. 程式碼生成與理解
- **範例：** "幫我建立一個計算費波納契數列的 Python 函數"
