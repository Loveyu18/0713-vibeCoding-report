什麼是 Claude Code？

Claude Code 是 Anthropic 推出的官方命令列工具，讓開發者能在終端機中直接與 Claude AI 互動

安裝與設定

Terminal 
# 安裝 Claude Code
$ curl -fsSL https://claude.ai/install.sh | sh

# 或使用 npm
$ npm install -g @anthropic-ai/claude-code

# 設定 API 金鑰
$ claude auth login

# 開始使用
$ claude "幫我寫一個 React 元件"

核心功能特色

  • 智能對話
    直接在終端機與 Claude 對話，獲得即時程式建議
  • 檔案操作
    讀取、分析、修改專案檔案，支援多種程式語言
  • 程式碼生成
    根據需求自動生成高品質的程式碼
  • 除錯協助
    分析錯誤訊息，提供修復建議
  • 重構優化
    改善程式碼結構，提升效能與可讀性

Claude Code Session
